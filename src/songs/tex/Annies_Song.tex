\v{} = innehalten \\
Das Lied hat John Dover als Ode an seine Frau Annie Martell Denver verfasst. Die Inspiration dazu sei ihm im Skilift auf dem Mountain Ajax gekommen, als ihn die Schönheit der Natur an sie denken lies. Daher habe er den Text innerhalb von 10,5 Minuten geschrieben. Zunächst sang er den Song auf den zweiten Satz von Tchaikowskys 5. Sinfonie. Als er eine Melodie dazu komponierte verblieben nur die ersten fünf Noten.
